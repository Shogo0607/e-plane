# Research Folder

Research is organized by region so each automation run can build on the existing notes instead of creating duplicate date folders.

- `research/north-america/notes.md`
- `research/europe/notes.md`
- `research/china/notes.md`
- `research/japan/notes.md`
- `research/summary.md`

Each run should:

1. Inspect the folder structure and read the current regional notes.
2. Perform fresh web search for North America, Europe, China, and Japan.
3. Compare new sources against the existing notes.
4. Append only new facts, changed regulatory positions, certification milestones, technical progress, or corrected/retired assumptions under a dated update heading.
5. Skip restating unchanged baseline material.
6. Put every source link in `/Volumes/ex/study/e-plane/MOC.md`.
