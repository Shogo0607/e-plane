# Research Folder

Use one dated folder per automation run:

- `research/YYYY-MM-DD/north-america.md`
- `research/YYYY-MM-DD/europe.md`
- `research/YYYY-MM-DD/china.md`
- `research/YYYY-MM-DD/japan.md`
- `research/YYYY-MM-DD/summary.md`

Each run should perform fresh web search before writing findings. Put all source links in `/Volumes/ex/study/e-plane/MOC.md`.
