# eVTOL・電動航空機コンポーネント要求スペックと計算方法

最終確認日: 2026-06-03

対象範囲: eVTOL、電動固定翼機、電動水上機、電動訓練機、将来の混合電気・水素電気航空機に必要な電池、モータ、インバータ、BMS、高電圧配電、熱管理、充電設備、冗長系の要求スペックと、その要求がなぜ必要になるかを計算で説明する。

## エグゼクティブサマリー

eVTOLと電動固定翼機では、必要スペックを支配する物理が異なる。eVTOLは垂直離着陸とホバリングを行うため、短時間に大きな出力を出せる電池、モータ、インバータが必要である。電動固定翼機は翼が揚力を発生するため巡航出力は小さいが、航続距離を伸ばすために高い電池比エネルギーが必要である。

大まかに言えば、eVTOLは「出力密度と熱」、電動固定翼は「エネルギー密度と巡航効率」が支配要件である。ただし、どちらも航空機であるため、熱暴走非伝播、絶縁、アーク故障、冗長性、故障時安全、継続耐空、ソフトウェア管理が自動車より厳しい。

中国の「低空航空器動力電池技術路線図 2025版」は、eVTOLに対してパック比エネルギー300Wh/kg以上、パック連続出力密度1600W/kg以上、4C連続・8Cピーク、1500サイクル以上を示す。電動固定翼にはパック比エネルギー400Wh/kg以上、パック連続出力密度1200W/kg以上、2C連続・4Cピーク、1000サイクル以上を示す。

NASA資料では、航空用電動機はMW級で13kW/kg、96%以上効率、将来はさらに高比出力・高効率が目標とされる。NASAの電力変換器例では250kW、1000VDC、99.3%効率、10.6kW/kgが示される。航空向け研究目標では、電動機14kW/kg超、パワーエレクトロニクス25kW/kg超、効率99%超、2kVまで部分放電なしが挙げられている。

### BMS

必要機能:

- セル電圧監視
- セル温度監視
- SOC推定
- SOH推定
- SOP、つまり出力可能電力の推定
- 絶縁監視
- 熱暴走予兆検出
- セルバランシング
- 接触器制御
- プリチャージ制御
- 急速充電制御
- EOL時の性能制限
- 飛行制御・エネルギー管理システムへの状態通知
- 整備用ログ記録

航空機ではSOCだけでは足りない。重要なのは、残量があるかではなく、その時点の温度、SOC、劣化状態で必要出力を安全に出せるかである。そのためSOP推定が重要になる。

### 高電圧配電

必要機能:

- 高電圧接触器
- プリチャージ回路
- ヒューズまたは半導体遮断器
- アーク故障検出
- 地絡・絶縁監視
- 高電圧インターロック
- 分離配線
- 冗長給電
- 故障区画の隔離
- 雷・静電・HIRF対策
- メンテナンス時の放電確認

eVTOLでは分散推進のため、高電圧を複数モータへ安全に分配する必要がある。単一バス故障が全推進喪失にならないよう、左右分割、前後分割、クロスフィード、複数バッテリーパック構成が検討される。

### 熱管理

対象:

- 電池
- モータ
- インバータ
- DC/DC
- 車載充電器または急速充電インターフェース
- キャビン空調

主な方式:

- 液冷プレート
- 冷却液ループ
- ラジエータ
- 誘電液浸漬
- 空冷
- 相変化材料
- 熱暴走ベント

熱管理は単なる冷却ではない。重量、空気抵抗、信頼性、漏液時の絶縁、整備性、火災時の排気経路まで含む機体設計である。

### 充電・地上設備

必要スペックは機体パック容量とターンアラウンド時間で決まる。

例:

- パック容量150kWh
- 80%分を15分で充電
- 理想電力 = 150kWh x 0.8 / 0.25h = 480kW
- 充電効率95%なら入力は約505kW

つまり、商用eVTOLや大型電動固定翼を高稼働で回すには、数百kW級の地上電力、冷却、消防、充電規格、課金、バッテリー健全性確認が必要である。

## なぜそのスペックが必要か

### eVTOLはホバー出力が支配する

eVTOLは垂直離着陸時に翼でなくロータで重量を支える。ホバーでは、機体重量を空気に運動量として与えて支えるため、大きな誘導出力が必要になる。

理想ホバー誘導出力:

```text
W = m g
DL = W / A
P_ideal = W * sqrt(DL / (2 rho))
```

記号:

- `m`: 機体質量 kg
- `g`: 重力加速度 9.81m/s2
- `W`: 重量 N
- `A`: 総ロータ円盤面積 m2
- `DL`: 円盤荷重 N/m2
- `rho`: 空気密度 kg/m3

実際の軸出力:

```text
P_shaft = P_ideal / FM + P_profile + P_accessory
```

記号:

- `FM`: figure of merit。ロータ効率を表す。概念設計では0.65から0.80程度。
- `P_profile`: ブレード断面抗力などのプロファイル損失
- `P_accessory`: 補機出力

式から分かること:

- 機体重量 `W` が増えると必要出力は強く増える。
- 総ロータ面積 `A` が小さい、つまり円盤荷重が高いと必要出力が増える。
- eVTOLで大径ロータや多数ロータが使われるのは、円盤荷重を下げるためである。

### 固定翼は巡航抗力が支配する

固定翼では、翼が揚力を作るため、ロータで重量を直接支える必要がない。巡航で必要なのは抗力に打ち勝つ出力である。

巡航抗力:

```text
q = 0.5 rho V^2
D = q S CD0 + k W^2 / (q S)
k = 1 / (pi e AR)
```

軸出力:

```text
P_shaft = D V / eta_prop
```

記号:

- `V`: 巡航速度 m/s
- `S`: 主翼面積 m2
- `CD0`: ゼロ揚力抗力係数
- `AR`: アスペクト比
- `e`: オズワルド効率
- `eta_prop`: プロペラ効率

式から分かること:

- 速度が高すぎると寄生抗力が増える。
- 速度が低すぎると誘導抗力が増える。
- 高揚抗比の翼は巡航エネルギーを減らす。
- 固定翼はホバーしないため、同じ重量でもeVTOLより航続を伸ばしやすい。

## サイジング手順

### 1. ミッションを分解する

まず飛行をフェーズに分ける。

eVTOLの例:

1. 地上待機
2. 垂直離陸・ホバー
3. 上昇
4. 遷移
5. 巡航
6. 降下
7. 遷移
8. 着陸ホバー
9. 予備
10. 緊急時、片系統故障、代替着陸

電動固定翼の例:

1. 地上待機
2. タキシング
3. 離陸滑走
4. 上昇
5. 巡航
6. 降下
7. 進入・着陸
8. 復行
9. 予備
10. 代替空港または滞空予備

各フェーズについて、出力 `P_i` と時間 `t_i` を置く。

### 2. 軸出力を電池出力へ換算する

```text
P_batt_i = P_shaft_i / (eta_motor * eta_inverter * eta_distribution)
```

例:

- モータ効率: 0.96
- インバータ効率: 0.985
- 配電効率: 0.99

```text
eta_total = 0.96 * 0.985 * 0.99 = 0.936
```

軸出力500kWが必要なら:

```text
P_batt = 500 / 0.936 = 534kW
```

### 3. ミッションエネルギーを計算する

```text
E_mission = sum(P_batt_i * t_i)
```

時間は時間単位にする。5分なら `5/60 = 0.0833h`。

予備を加える:

```text
E_required = E_mission + E_reserve
```

予備には以下を含む。

- 法規上の予備
- 代替着陸
- 風
- 低温
- 経路変更
- 着陸待ち
- 電池劣化
- SOC下限

### 4. 公称電池容量を決める

航空機では、電池を0%まで使う設計はしない。低SOCでは出力が落ち、劣化も進む。さらに寿命末期EOLでもミッションを満たす必要がある。

```text
E_nameplate = E_required / (usable_SOC_window * EOL_capacity_fraction)
```

例:

- 必要実エネルギー: 80kWh
- 使用SOC幅: 70%
- EOL容量: 80%

```text
E_nameplate = 80 / (0.70 * 0.80) = 143kWh
```

これが、実ミッション80kWhでも150kWh級パックが必要になる理由である。

### 5. 電池重量をエネルギー制約で計算する

```text
m_batt_energy = E_nameplate / specific_energy_pack
```

例:

- 公称容量: 150kWh
- パック比エネルギー: 300Wh/kg = 0.300kWh/kg

```text
m_batt_energy = 150 / 0.300 = 500kg
```

### 6. 電池重量を出力制約で計算する

```text
m_batt_power = P_peak / specific_power_pack
```

例:

- ピーク電池出力: 800kW
- パック出力密度: 1.6kW/kg

```text
m_batt_power = 800 / 1.6 = 500kg
```

必要な電池重量:

```text
m_batt = max(m_batt_energy, m_batt_power)
```

eVTOLでは出力制約が支配しやすい。電動固定翼ではエネルギー制約が支配しやすい。

### 7. Cレートを確認する

```text
C_rate = P_batt(kW) / E_nameplate(kWh)
```

例:

- 電池容量150kWh
- ピーク出力600kW

```text
C_rate = 600 / 150 = 4C
```

150kWhで1200kWを出すなら8Cである。eVTOLが4C連続、8Cピーク級を要求される理由はこの関係で説明できる。

### 8. モータをサイジングする

総必要軸出力をロータ/プロペラ数で割る。

```text
P_motor_peak = P_total_peak / N_available * margin
```

`N_available` は故障を考慮した有効モータ数である。

例:

- 総ピーク軸出力: 900kW
- モータ数: 6
- 1基故障時も安全着陸したい
- 有効モータ数: 5
- 余裕: 1.1

```text
P_motor_peak = 900 / 5 * 1.1 = 198kW
```

つまり通常時なら1基150kWで足りても、故障時要件を入れると200kW級が必要になる。

連続定格は、最も長い高出力フェーズで熱平衡を満たす値にする。ピーク出力だけでモータを選ぶと、上昇やホバーで熱的に成立しない。

### 9. インバータと電流を計算する

DC電流:

```text
I_DC = P / (V_DC * eta_inverter)
```

例:

- 出力400kW
- インバータ効率0.985

800Vの場合:

```text
I = 400000 / (800 * 0.985) = 508A
```

400Vの場合:

```text
I = 400000 / (400 * 0.985) = 1015A
```

配線損失:

```text
P_loss_wire = I^2 R
```

電圧を2倍にすると電流は半分になり、同じ抵抗ならI2R損失は4分の1になる。このため、航空電動推進では800Vから1kV超、将来は2kV級が検討される。

### 10. 熱損失を計算する

出力機器の損失:

```text
Q_loss = P_out * (1 / eta - 1)
```

例:

- モータ軸出力500kW
- 効率96%

```text
Q_loss = 500 * (1/0.96 - 1) = 20.8kW
```

500kWモータ1基で約21kWの熱が出る。6基なら同時ピークで100kW超の熱になる可能性がある。熱管理重量が無視できない理由である。

冷却液流量:

```text
m_dot = Q_loss / (cp * DeltaT)
```

例:

- 損失20kW
- 冷却液比熱 `cp = 3500J/kg/K`
- 温度上昇 `DeltaT = 10K`

```text
m_dot = 20000 / (3500 * 10) = 0.57kg/s
```

この流量をポンプ、配管、ラジエータ、熱交換器で成立させる必要がある。

## 例: 5人乗りeVTOLの概算

前提:

- 最大離陸重量: 2500kg
- 総ロータ円盤面積: 50m2
- 空気密度: 1.225kg/m3
- figure of merit: 0.75
- 電動系効率: 0.936

重量:

```text
W = 2500 * 9.81 = 24525N
DL = 24525 / 50 = 490.5N/m2
P_ideal = 24525 * sqrt(490.5 / (2 * 1.225)) = 347kW
P_shaft_hover = 347 / 0.75 = 463kW
P_batt_hover = 463 / 0.936 = 495kW
```

この値は理想化されたホバー誘導出力中心の概算であり、実際にはプロファイル損失、上昇、遷移、制御余裕、故障時余裕、暑熱・高地条件で増える。

もし安全余裕込みのピーク電池出力を900kWとし、電池容量を150kWhとすると:

```text
C_rate_peak = 900 / 150 = 6C
```

したがって、eVTOLでは8Cピーク級が現実的な要求になる。

## 例: 4座電動固定翼の概算

前提:

- 最大離陸重量: 1260kg
- 巡航速度: 140km/h = 38.9m/s
- 揚抗比 L/D: 14
- プロペラ効率: 0.82
- 電動系効率: 0.936

巡航抗力:

```text
W = 1260 * 9.81 = 12361N
D = W / (L/D) = 883N
P_shaft = D * V / eta_prop = 883 * 38.9 / 0.82 = 41.9kW
P_batt = 41.9 / 0.936 = 44.8kW
```

1.5時間巡航相当:

```text
E_cruise = 44.8 * 1.5 = 67.2kWh
```

これはRX4Eの70kWh級電池容量と近いオーダーになる。実際には離陸・上昇・予備・SOC下限・EOLを考慮するため、認証上の最大航続や運用上の実航続は慎重に見る必要がある。

## コンポーネント別の設計チェックリスト

### 電池セル選定

- 比エネルギーは目標ミッションを満たすか。
- 必要CレートをEOL、低SOC、高温、低温で満たすか。
- 高出力時の温度上昇は許容範囲か。
- 熱暴走時に隣接セル・隣接モジュールへ伝播しないか。
- 急速充電で寿命が許容範囲か。
- セルばらつきと劣化ばらつきをBMSで管理できるか。
- 航空用品質管理、トレーサビリティ、ロット管理が可能か。

### 電池パック設計

- セルからパックへの重量効率が高いか。
- 冷却構造が均一で軽いか。
- モジュール単位で火炎、ガス、破片を封じ込められるか。
- ベント経路が乗員、構造、推進系に危険を与えないか。
- 交換式ならロック、誤挿入防止、絶縁、冷却接続が安全か。
- クラッシュ荷重と着陸衝撃に耐えるか。
- 継続耐空で容量・内部抵抗・イベント履歴を管理できるか。

### モータ

- ピーク出力と連続出力を分けて評価しているか。
- 熱平衡で連続運転できるか。
- 磁石の減磁、巻線絶縁、軸受寿命を満たすか。
- プロペラ/ロータ回転数とトルクに適合しているか。
- 減速機を使う場合、重量、効率、騒音、信頼性を評価したか。
- 片系統故障時に安全着陸できるか。

### インバータ

- 必要電圧・電流に対して余裕があるか。
- SiCデバイスの短絡耐量、ゲートドライブ保護、dv/dt対策が十分か。
- 冷却喪失時のディレーティングが定義されているか。
- EMIが飛行制御、通信、航法、センサへ悪影響を与えないか。
- ソフトウェアと制御則が認証可能か。
- 地絡、アーク、過温、過電圧時に安全に遮断できるか。

### 高電圧配電

- 単一故障で全推進喪失しないか。
- 配線分離、火災区画、遮断器配置は妥当か。
- プリチャージと放電確認は整備手順に入っているか。
- 高高度低圧で部分放電しないか。
- 雷・静電・HIRFの影響を評価したか。

### 熱管理

- 最悪日、低速、ホバー、急速充電で熱収支が成立するか。
- ラジエータ抵抗が機体性能を悪化させないか。
- 冷却液漏れで絶縁・火災リスクが増えないか。
- ポンプ故障時のディレーティングと緊急着陸が成立するか。
- 熱暴走時のベント方向と乗員保護は妥当か。

## eVTOLと電動固定翼の設計思想の違い

### eVTOL

重点:

- 高出力密度電池
- 高Cレート
- 多発分散推進
- 冗長飛行制御
- ホバー熱管理
- 低騒音ロータ
- 故障時安全着陸
- 充電・高稼働率

eVTOLは短距離で高頻度に飛ばす事業モデルと相性がある。一方、垂直離着陸はエネルギーを多く使うため、長距離化には不利である。

### 電動固定翼/eCTOL

重点:

- 高比エネルギー電池
- 高揚抗比
- 軽量構造
- 巡航効率
- 既存空港・通用空港活用
- 訓練・観光・短距離移動

電動固定翼はeVTOLよりエネルギー効率が良い。ただし滑走路や水域が必要で、都市内ドアツードア性は低い。実用化は訓練、遊覧、短距離固定ルートから進みやすい。
