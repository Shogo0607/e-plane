# 中国の非eVTOL電動航空機調査

最終確認日: 2026-06-03

対象範囲: 中国で開発、認証、量産、導入、または中国拠点化が確認できる電動航空機のうち、eVTOLを主目的としないもの。主に固定翼、通常離着陸機、軽量スポーツ機、水上電動機、訓練機、将来の混合電気・水素電気固定翼を扱う。eVTOLは比較対象として必要な場合だけ触れる。

## エグゼクティブサマリー

中国の非eVTOL電動航空機は、2026年6月時点では大型旅客機や地域航空機ではなく、2座から4座の小型固定翼機が実用化の中心である。最も実績があるのは沈陽航空航天大学系の遼寧通用航空研究院が開発した「鋭翔 RX」シリーズで、RX1E、RX1E-A、RX1E-S、RX4Eが代表機種である。

量産・商業化では、零重力飛機工業と傘下・投資先の鋭恩飛機工業がRX1E-Aを扱っている。RX1E-Aは中国で商業運航可能な電動固定翼機として扱われ、飛行訓練、体験飛行、低空観光、巡視用途が中心である。

認証上の大きな転換点は、2022年のCCAR-23改正である。中国民航局 CAAC は「正常類飛機適航規定」に電動航空機の動力装置補充要求であるH章を追加し、電推系、電池・配電系、電池・電動力系防火を明示した。RX4Eはこの枠組みでCCAR-23正常類純電動機として型式証明を取得した。

技術的には、短距離・小型用途では純電動が成立しつつあるが、航続距離とペイロードは電池比エネルギーに強く制約される。中国国内では航空用電池として300Wh/kg超、将来400から500Wh/kg級を目指す動きがある。正力新能などの電池企業が航空電池の適航・品質認証を取り始めており、車載電池産業の航空転用が進む。

## 主要プレイヤー

### 遼寧通用航空研究院 / 沈陽航空航天大学

中国の有人電動固定翼機で最も重要な研究・開発主体である。中国工程院院士の楊鳳田氏のチームを中心に、鋭翔RXシリーズを開発してきた。

主な役割:

- RX1E、RX1E-A、RX1E-S、RX4Eの研究開発
- 電動軽量スポーツ機、電動水上機、4座正常類電動機の適航取得
- 中国の電動航空機審査実務における先行事例の形成

重要性:

- RX1Eは中国初期の型式設計承認・生産許可を得た電動軽量スポーツ機である。
- RX1E-Aは商業運航可能な2座電動固定翼機として位置付けられる。
- RX4EはCCAR-23正常類の4座純電動機として型式証明を得た。

### 零重力飛機工業 / 鋭恩飛機工業

零重力飛機工業はeVTOLでも知られるが、非eVTOL領域ではRX1E-Aの商業化、量産、納入で重要である。鋭恩飛機工業（浙江）はRX1E-Aの生産許可を持つ主体として報じられている。

主な役割:

- RX1E-Aの生産・販売・商業納入
- 低空観光、初級飛行訓練、体験飛行、巡視用途への展開
- 正力新能など電池サプライヤーとの連携

2025年3月31日に首批商業運営電動固定翼機としてRX1E-Aを納入したと報じられた。中新網は、同機を中国で唯一、型式合格証と生産許可を取得し商業化運航可能な電動固定翼機と説明している。

### 正力新能

航空電池サプライヤーとして重要である。RX1E系列向けの航空電池を供給し、AS9100D航空宇宙品質管理体系認証および航空電池適航関連認証を取得したとされる。

公表値:

- 航空電池システムの能量密度: 320Wh/kg超
- 15分で80%充電
- 20%低電量状態でも12C超放電可能
- 材料、セル、システムの国産化を訴求

この性能は、電動固定翼の短時間高出力と急速補電の両方を狙ったものである。飛行訓練や観光用途では、機体性能だけでなく日次稼働率が事業性を決めるため、急速充電または5分程度の交換電池が重要になる。

### 江蘇晟翼航空 / Elektra Solar

ドイツのElektra Solar GmbHと蘇宿工業園区の協力により、中国R&D・製造拠点として江蘇晟翼航空が設立された。第一段階の総装下線機種はElektra Trainerとされる。

Elektra Trainerはドイツ側の技術を基礎にした2座電動固定翼訓練機である。中国国内のCAAC型式証明取得は確認できていないため、RXシリーズのような中国認証済み機体とは分けて扱う必要がある。

公表値:

- 標準電池: 30kWh
- 航続時間: 2.5時間
- 航続距離: 最大300km
- 充電: 空電池から約3時間、1時間充電で1時間飛行を訴求

想定用途:

- 景区遊覧
- 低空通航人材訓練
- 低空巡検
- 通信中継など

### Yuneec / 昊翔

Yuneecは中国発の早期電動有人機開発企業として歴史的に重要である。E430、eSpyder、EVivaなどが知られる。ただし2026年時点で、中国の商用電動固定翼機の主流プレイヤーとは言いにくい。

E430は2座の電動軽飛行機として開発され、48kW級電気推進システムと炭素繊維複合材機体を訴求した。ドイツ超軽量機認証や米国市場展開の動きはあったが、大規模な量産・中国国内商業運航の中心にはなっていない。

## 主要機種

### RX1E

位置付け:

- 中国初期の2座電動軽量スポーツ機
- RXシリーズの基礎機体
- 訓練、航空スポーツ、低空体験飛行向け

開発・認証:

- 2012年開発開始
- 2013年6月初飛行
- 2015年2月6日にCAACの型式設計承認 TDA を取得
- 2015年11月25日に生産許可 PC を取得

特徴:

- 2座横並び
- 複合材構造
- 電動軽量スポーツ機
- 初期型は航続45分級とされ、商業運航には短い

評価:

RX1Eは中国電動固定翼の基礎となった機体である。商業性は限定的だが、設計、製造、試験、試飛、適航審査の一連の経験を中国国内に蓄積した点が大きい。

### RX1E-A

位置付け:

- RX1Eの増程型
- 中国で商業運航可能な2座電動固定翼機として扱われる
- 初級飛行訓練、体験飛行、低空観光、低空巡視向け

開発・認証:

- 2017年11月1日初飛行
- 2018年10月19日に型式合格証 TC を取得
- 2019年4月11日に生産許可 PC を取得
- 2025年に商業運営向け初批納入が報じられた

主要スペック:

| 項目 | 値 |
| --- | --- |
| 座席 | 2名 |
| 最大離陸重量 | 630kg |
| 電推システム最大出力 | 50kW |
| 航続時間 | 150分 |
| 航続距離 | 280km |
| 巡航速度 | 120km/h |
| 交換電池時間 | 約5分 |
| 安全装備 | 機体用パラシュート選択可 |

技術的意味:

RX1E-AはRX1Eから最大出力を40kWから50kWへ、最大離陸重量を500kgから630kgへ増やし、航続を45分から150分へ伸ばした。商業運航に必要な最小限の飛行時間を確保したことが重要である。

評価:

電動航空機の初期市場では、旅客輸送よりも訓練と観光の方が成立しやすい。RX1E-Aは、電動固定翼が低騒音、低運航費、短距離反復運航に向くことを利用した機体である。ただし速度と航続は従来の燃料式軽飛行機より低く、広域移動用途には不向きである。

### RX1E-S

位置付け:

- RX1E-Aを基礎にした2座電動水上機
- 水資源が豊富で環境保護要求の高い景区・生態観光・水域監視向け

開発・認証:

- 2019年初飛行
- 2021年12月31日に型式合格証 TC を取得
- 2022年に生産許可 PC 取得と報じられる

主要スペック・特徴:

| 項目 | 値・特徴 |
| --- | --- |
| 座席 | 2名 |
| 構成 | 双フロート式水上電動機 |
| 電動機最大出力 | 80kW |
| プロペラ | 炭素繊維3枚プロペラ |
| 最大回転数 | 2600rpm |
| フロート排水量 | 1.2立方メートル |
| 構造 | 炭素繊維複合材フロート、複数水密区画 |
| 操縦補助 | 可収放水舵 |

評価:

RX1E-Sは非eVTOL電動機の用途開拓として重要である。水上機は通常の滑走路がなくても水域を使えるため、景区観光、漁政、環境監視、河川・湖沼巡視に適する。一方で、水面離着水の安全、波高、浮体の耐久、腐食対策、救難体制、桟橋・充電設備が実運用の制約になる。

### RX4E

位置付け:

- 中国初の4座純電動正常類固定翼機
- CCAR-23正常類で型式証明を得た電動航空機
- 訓練、私用飛行、短距離移動、観光、写真測量向け

開発・認証:

- 2018年から工信部支援で開発開始
- 2019年10月28日に検証機初飛行
- 2019年11月11日にCAACが型式合格申請を受理
- 2024年末に型式合格証を取得
- 2025年1月2日に中国政府網・新華社が公表

主要スペック:

| 項目 | 値 |
| --- | --- |
| 座席 | 4名 |
| 認証カテゴリ | CCAR-23正常類 |
| 翼幅 | 13.5m |
| 全長 | 8.4m |
| 最大離陸重量 | 1260kg |
| 電池容量 | 70kWh |
| 電推システム最大出力 | 140kW |
| 航続時間 | 最大1.5時間 |
| 航続距離 | 約300km |

技術的意味:

RX4Eは中国で「電動軽量スポーツ機」から「正常類小型機」へ進む節目である。CCAR-23は19席以下、最大証明離陸重量8618kg以下の小型正常類機を扱う枠組みであり、RX4Eはその電動版の先行事例になった。

評価:

4座化により訓練だけでなく、短距離移動や遊覧の商業性が増す。しかし70kWh、1.5時間、300kmという値から分かるように、現状の純電動固定翼はまだ短距離用途である。派生として水上型、雪上型、水素動力型、特種用途型が計画されており、純電動から水素電気・混合電気へ広げる可能性がある。

位置付け:

- 中国企業がドイツ技術を用いて中国拠点化する2座電動訓練機
- 中国低空経済向けの訓練・観光・巡検を狙う

主要スペック:

| 項目 | 値 |
| --- | --- |
| 座席 | 2名 |
| 標準電池 | 30kWh |
| 航続時間 | 2.5時間 |
| 航続距離 | 最大300km |
| 充電 | 空電池から約3時間 |

評価:

RXシリーズと比較すると、CAAC認証済みの中国国産機というより、海外技術の中国展開という位置付けである。低空経済の需要拡大に応じて、中国国内で認証、製造、訓練事業が進むかが焦点である。

## 規制・認証動向

### CCAR-23 H章の追加

2022年5月6日に改正公表され、2022年8月1日から施行されたCCAR-23-R4では、電動航空機向けにH章「電動飛機動力装置補充要求」が追加された。

H章の主要対象:

- 電推システム
- 電池および配電システム
- 電池および電動力システムの防火

意義:

- 電動航空機を従来の燃料式動力装置の延長でなく、電池、配電、電推、防火を含むシステムとして審査する枠組みを作った。
- RX4Eのような正常類電動固定翼機の型式証明に直接関係する。
- 中国が電動小型機の適航標準を国際的に先行整備する狙いを持つ。

### 型式証明、製造、運航の関係

電動航空機の実用化では、単に型式合格証 TC を取るだけでは不十分である。

必要な段階:

1. 型式合格証 TC: 設計が適航基準に適合すること。
2. 生産許可 PC: 製造品質システムが適合設計を反復生産できること。
3. 単機の適航証 AC: 個々の機体が飛行可能な状態にあること。
4. 運航者・運航許可: 商業運航、訓練、観光、遊覧、巡視などの運用を実施できること。
5. 継続耐空: 電池交換、容量劣化、BMSデータ、整備周期、ソフト更新、熱管理部品交換などを管理すること。

RX1E-AはTCとPCを得て商業納入に進んでいる。RX4EはTC取得が大きな節目だが、量産・商業運航拡大にはPC、AC、運航体制、整備体制の確認が必要である。

### 低空経済政策との関係

中国では、低空経済が国家戦略・成長産業として扱われている。CAACは通用航空と低空経済に関する作業体制を強化し、2026年5月には低空安全司を設置した。

重点領域:

- 低空民航発展計画
- 低空安全と発展の調整
- 低空飛行サービス派遣平台
- 飛行サービスステーション体系
- 適航審定
- 市場監督
- 運航監督
- 安全監督体系

非eVTOL電動固定翼にとっては、低空観光、訓練、巡検、水域監視、低空公共サービスが直接の市場になる。eVTOLほど都市内垂直離着陸に依存しないため、既存空港、通用空港、水上運航区、臨時離着陸場の整備が鍵になる。

### 無人航空機制度との関係

無人固定翼や大型貨物UASは、2024年1月施行の「無人駕駛航空器飛行管理暫行条例」の対象となる。同条例では無人航空機を微型、軽型、小型、中型、大型に分類し、中型・大型民用無人航空機の設計、生産、輸入、飛行、整備には適航許可が必要とされる。

重量区分:

- 中型: 最大離陸重量150kg以下、ただし微型・軽型・小型を除く
- 大型: 最大離陸重量150kg超

本調査の主対象は有人固定翼だが、将来の電動固定翼貨物機や電動無人輸送機はこの制度の影響を受ける。

## 技術動向

### 電池

電動航空機最大の制約は電池である。航空燃料は重量あたりのエネルギーが非常に高く、電池は電動パワートレイン効率の高さを考慮しても航続距離で不利である。

現状の論点:

- 現在の航空用リチウムイオン電池パックは、実装状態で200Wh/kg前後から300Wh/kg超へ進む段階。
- 中国国内の航空電池では、正力新能が320Wh/kg超を公表している。
- 将来の目標として、セル400から500Wh/kg、パック300から400Wh/kg超が重要。
- 固体電池、半固体、凝聚態、リチウム金属などが注目されるが、航空認証可能な安全性と量産性が課題。

電動固定翼に必要な電池性能:

| 指標 | 目安 | 理由 |
| --- | --- | --- |
| パック比エネルギー | 400Wh/kg以上が望ましい | 航続距離とペイロードを支配するため |
| セル比エネルギー | 500Wh/kg以上が望ましい | パック化で筐体・冷却・BMS分の重量が増えるため |
| 連続放電 | 2C以上 | 離陸・上昇・巡航を余裕を持って支えるため |
| ピーク放電 | 4C以上 | 離陸、復行、緊急時出力に備えるため |
| サイクル寿命 | パック1000回以上、セル1200回以上 | 訓練・観光で反復運航するため |
| 充電/交換 | 15分以内急速充電または5分交換 | 稼働率が事業性を決めるため |
| 安全 | 熱暴走非伝播、過充電/過放電保護、絶縁監視 | 航空事故防止と認証のため |

### モータ・電推システム

中国の公開値では、RX1E-Aが50kW、RX1E-Sが80kW、RX4Eが140kW級である。これは小型固定翼としては妥当な範囲で、従来の小型ピストンエンジンの代替を狙う出力である。

重要指標:

- 比出力 kW/kg
- 効率
- 連続定格とピーク定格の差
- 冷却方式
- プロペラ適合
- インバータとの統合
- 故障時挙動
- EMI/HIRF/雷対策
- 継続耐空文書

小型電動固定翼では、モータ自体の比出力よりも、電池重量と航続距離の方が厳しい制約になる。ただし多座化、長航続化、将来の水素電気・混合電気化では、モータ、発電機、インバータの高比出力化が不可欠になる。

### インバータ・高電圧配電

電動航空機は高出力を扱うため、低電圧では電流が大きくなり、配線、バスバー、接触器、冷却、損失が重くなる。4座機や将来の大型化では800V級から1kV超の直流母線が重要になる。

設計論点:

- SiCパワーデバイス
- 高効率スイッチング
- 絶縁監視
- アーク故障検出
- プリチャージ
- 短絡保護
- 高高度・低圧での部分放電
- 冷却
- 高電圧安全整備手順

### 熱管理

電動航空機では、電池、モータ、インバータ、充電器がすべて熱源になる。特に短時間高出力、急速充電、夏季低空観光、連続訓練では熱管理が事業性を左右する。

重点:

- 電池温度の均一化
- 高C放電時の発熱抑制
- 急速充電時の冷却
- 熱暴走時の隔離・ベント
- モータ巻線と磁石温度管理
- インバータパワーモジュール温度管理
- 冷却液漏れと絶縁安全

### 機体設計

小型電動固定翼では、高揚抗比、軽量構造、複合材、プロペラ効率、バッテリー搭載位置、重心管理が重要である。燃料機と違い、電池重量は飛行中に減らない。そのため離陸重量と着陸重量がほぼ同じであり、着陸装置、構造、失速速度、非常時エネルギー管理に影響する。

電動固定翼で重要な設計方針:

- 低速巡航で高効率な翼
- 軽量複合材構造
- 電池搭載位置を重心近くに置く
- 交換電池の場合は構造・接続・冷却・ロック機構を安全にする
- 低騒音プロペラ
- 訓練用途では整備性と堅牢性を優先する

## 商業化用途

### 飛行訓練

最も現実的な用途である。理由は、飛行時間が短く、同じ空港に戻り、充電・交換が容易で、低騒音の価値が高いからである。初級訓練では1ソーティーあたり1時間前後の飛行が多く、RX1E-Aの150分級航続は商業上の余裕を持つ。

### 低空観光・体験飛行

景区や水域での短距離遊覧は、電動機の低騒音・低排出と相性が良い。RX1E-A、RX1E-S、Elektra Trainerが狙う市場である。ただし観光用途では安全説明、保険、天候判断、乗客導線、地上充電、騒音規制、自治体許可が必要になる。

### 巡視・測量

送電線、河川、湖沼、森林、漁政、環境監視では、低運航費と低騒音が利点になる。ただし有人機で行う必要があるか、無人固定翼やマルチローターで足りるかは用途ごとに変わる。

### 短距離移動

RX4Eのような4座機では、短距離移動への応用余地がある。ただし、300km級の公称航続は予備、気象、乗員重量、電池劣化を考慮すると実運航距離はより短くなる。定期旅客輸送としてはまだ限定的である。

## 課題

### 電池重量

電動航空機は燃料機と違って、飛行中にエネルギー使用分の重量が減らない。長距離化しようとすると電池重量が増え、その重量を支えるためにさらにエネルギーが必要になる。これが純電動航空機の根本制約である。

### 適航と安全

電池熱暴走、絶縁破壊、アーク、BMS異常、急速充電劣化、ソフトウェア更新、冷却故障など、燃料機と異なるリスクがある。CAACはCCAR-23 H章で制度化を進めたが、実運航データはまだ少ない。

### 経済性

電気代は安くても、電池交換費、寿命、充電設備、稼働率、整備士訓練、保険、初期機体価格が事業性を左右する。観光・訓練用途では成立しやすいが、広域輸送ではまだ難しい。

### インフラ

通用空港、飛行サービス站、水上運航区、充電設備、交換電池設備、消防、救難、整備設備が必要である。eVTOL用バーティポートとは異なるが、低空飛行サービス体系との接続は共通課題である。
